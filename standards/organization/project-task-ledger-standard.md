# Project Task Ledger Standard

The project task ledger records project-level work history inside the central project record.

## Location

Default file:

`workspace/projects/<project-id>/memory/task-ledger.md`

## Required Fields

Each entry should include:

- Date.
- Task id or short name.
- User request summary.
- Owner Pal judgement.
- Execution layer used, if any.
- Files or artifacts touched.
- Verification status.
- Follow-up items.

The task ledger is not a substitute for evidence. It points to task packages, verification records, and reports when those exist.
