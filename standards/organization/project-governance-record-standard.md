# Project Governance Record Standard

Project governance records describe rule changes, workflow improvements, and Pal performance observations for a bound external project.

## Location

Default location:

`workspace/projects/<project-id>/governance/`

## Allowed Records

- Rule change proposals.
- Workflow improvement notes.
- Pal performance reviews.
- Project-specific boundary decisions.

## Boundary

Governance records belong to AgentPal's central project record. They must not be silently written into the external user project. If a user wants governance material inside their project, create a user-facing business document only after an explicit request.
