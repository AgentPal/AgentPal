# Governance Standards

This directory is the standards landing area for AgentPal governance records.

Governance records include review decisions, workflow improvement proposals, Pal performance reviews, and rule-change proposals. Real project governance records belong in the central project record under `workspace/projects/<project-id>/governance/`, not in an external project's `.agentpal/` directory.

This directory stores public standards only. It must not contain private project facts, internal development reports, or runtime-local state.
