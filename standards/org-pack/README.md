# Organization Pack Standards

This directory is the standards landing area for future AgentPal organization pack rules.

An org pack is a no-code package of public-safe organization assets such as Pal team definitions, workflows, templates, and governance conventions. It is not a runtime plugin, installer, marketplace package, or automatic execution layer.

Official reusable org-pack assets belong under `official/org-packs/`. User or project-specific org records belong in central workspace records when approved.
