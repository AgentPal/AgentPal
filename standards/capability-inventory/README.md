# Capability Inventory Standards

This directory is the standards landing area for capability inventory governance.

Capability inventory records describe runtime, model, Skill, plugin, MCP, Pal, and business-system availability as auditable facts or unknowns. They are not automatic scans and they do not install or enable capabilities.

Current profile examples remain under `capabilities/`. Project-specific capability records belong under the selected central project record at `workspace/projects/<project-id>/capability-inventory/`.
