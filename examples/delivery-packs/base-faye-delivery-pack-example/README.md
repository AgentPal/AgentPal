# Base Faye Delivery Pack Example

This is a public-safe example shell for a Faye-authored Delivery Pack.

It demonstrates the minimum public shape only. It does not include real customer data, private project evidence, credentials, connector settings, runtime execution, marketplace logic, scanner logic, validator programs, keyword routing, central roster changes, official Pal changes, or external project `.agentpal/` payloads.

## Scenario

Faye receives a fictional business goal:

```text
Help a small content team turn one product campaign into a repeatable AI-assisted delivery workflow.
```

Faye would prepare:

- a Delivery Pack using the `Project -> Flow Pack -> Task Package` structure;
- a Pal Team Blueprint with candidate perspectives;
- a Faye Build Request for PalSmith;
- a first Task Package after missing information is resolved.

## Files

- `README.md`
- `delivery-pack.json`

This example intentionally stays small because R132-A establishes the standard and base template. Filled realistic examples should be created in a separate approved round.

## Boundary

Candidate Pals are AI judgement inputs only. PalSmith remains responsible for Pal creation or completion. Faye does not modify central contacts, official Pals, external project bindings, tags, releases, or remote state.
