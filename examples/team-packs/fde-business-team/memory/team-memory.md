# FDE Business Team Memory

This public example stores memory policy only.

## Allowed Memory Candidates

- reusable interview question patterns;
- public-safe process discovery lessons;
- generic team handoff patterns;
- routing lessons about when Faye should exit.

## Not Allowed

- confidential interview notes;
- real customer data;
- revenue, contract, credential, or internal operating records;
- private user memory;
- unpublished business strategies.

## Writeback Policy

Writeback is a candidate only after Closure Gate and only if the content is public-safe and useful for future FDE judgement.
