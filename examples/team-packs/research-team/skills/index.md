# Research Team Skills Index

Team-level Skills here describe shared methods. They do not replace Vega,
Nova, Quinn, Mira, or any Pal-owned Skill.

## Available Team Methods

- `research-question-framing`: turn a broad topic into focused research questions.
- `source-inventory-discipline`: keep source availability and source limits visible.
- `evidence-synthesis`: organize findings by claim, evidence, confidence, and uncertainty.
- `reviewable-brief-writing`: write a brief that a verifier can inspect without guessing hidden work.

## Runtime Boundary

This Team Pack does not browse, scrape, run tools, or call APIs by itself. Runtime tools may be used only when the current host provides evidence and the selected Pal requests them.
