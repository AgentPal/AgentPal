# Marketing Growth Team Memory

This public example stores only safe memory rules and candidate patterns.

## Allowed Memory Candidates

- reusable AgentPal positioning angles;
- public-safe topic patterns;
- channel lessons that do not include private analytics;
- claim-boundary lessons about current vs future AgentPal capabilities;
- visual brief patterns that do not include private assets.

## Not Allowed

- account credentials;
- private analytics exports;
- customer lists;
- unpublished partner plans;
- private campaign budgets;
- copied third-party content.

## Writeback Policy

Writeback is proposed only after Closure Gate. Public examples may record
abstract learning candidates, not private campaign evidence.
