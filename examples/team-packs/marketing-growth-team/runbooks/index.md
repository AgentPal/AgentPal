# Marketing Growth Team Runbook Index

## Weekly Promotion Procedure

1. Confirm campaign goal, audience, channels, and constraints.
2. Select weekly content themes and topic rationale.
3. Draft five content items with channel notes.
4. Produce visual requirements for each item.
5. Review brand risk, evidence needs, and future-feature claims.
6. Prepare publishing plan and owner handoff.
7. After publication evidence exists, summarize performance and writeback candidates.

## Exception Handling

- If the user asks for actual publication, create an external execution Step and require evidence.
- If AgentPal feature claims are uncertain, ask Vega or Quinn to review the claim boundary.
- If a visual role is required and no owner exists, keep `visual-brief-owner` as an open role or ask the user whether to create a named Pal later.
