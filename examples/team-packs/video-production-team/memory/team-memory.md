# Video Production Team Memory

This public example stores memory policy only.

## Allowed Memory Candidates

- reusable video brief structures;
- common claim-review issues;
- public-safe production checklist improvements;
- generic platform-format lessons.

## Not Allowed

- private media assets;
- customer footage;
- licensed music or image files;
- unpublished campaign plans;
- private user preferences unless explicitly approved for the correct private memory scope.

## Writeback Policy

Writeback is proposed only after Closure Gate and only as a public-safe candidate.
