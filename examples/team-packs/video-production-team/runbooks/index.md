# Video Production Team Runbook Index

## Production Brief Procedure

1. Confirm goal, audience, format, platform, and duration.
2. Define message hierarchy and proof requirements.
3. Draft script and scene structure.
4. List visual style, required assets, and accessibility needs.
5. Review claims, missing assets, and readiness.
6. Produce handoff brief for a human editor or runtime if authorized.

## Exception Handling

- If the user asks for actual rendered video, mark rendering as an external execution Step.
- If assets are missing, keep the Step open as `blocked` or close it as `skipped` with a reason.
- If claims cannot be sourced, mark them as unverified or remove them from the script.
