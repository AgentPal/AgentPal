# Base AgentPal Org Pack Example

This is a public-safe example Org Pack for AgentPal.

It demonstrates the minimum shape of a reusable no-code organization package. It contains no real customer data, no credentials, no connector settings, no scanner, no validator, no installer, no marketplace program, and no keyword routing.

## Files

- `ORG.md`
- `org.json`
- `recommended-pals.md`
- `thin-binding-policy.md`
- `capability-inventory-policy.md`
- `governance-policy.md`
- `verification-checklist.md`

## Boundary

This example does not copy central Pal contacts, does not modify the central roster, and does not create external project `.agentpal/org-pack/`.
