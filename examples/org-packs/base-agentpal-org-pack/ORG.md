# Base AgentPal Org Pack Example

## Purpose

This example shows a small reusable AgentPal organization package for teams that want central Pal coordination, thin project binding, capability governance, and explicit verification.

## Reusable Assets

- public-safe organization purpose
- recommended Pal perspectives
- thin binding policy
- capability inventory policy
- governance policy
- verification checklist

## Customer-Private Assets

This example includes no customer-private assets.

Customer data, customer credentials, real project memory, and private evidence belong outside this public example.

## Routing Boundary

Recommended Pals are AI judgement inputs only. They are not keyword routes, domain routes, role routes, or fixed assignments.
