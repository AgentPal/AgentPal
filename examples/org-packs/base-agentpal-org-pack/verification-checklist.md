# Verification Checklist

Use this checklist before treating this Org Pack example as reusable:

- [ ] `org.json` parses.
- [ ] Recommended Pals are AI judgement inputs only.
- [ ] No central roster copy exists.
- [ ] No external project `.agentpal/org-pack/` write is required.
- [ ] No credentials or customer-private data are present.
- [ ] No connector, scanner, validator, installer, marketplace program, daemon, database, auto sync, or auto execution engine is included.
- [ ] No keyword routing, domain routing, role routing, or deterministic Pal assignment is included.
- [ ] Customer-private and reusable assets are separated.
