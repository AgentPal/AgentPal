# User Guide Examples

This directory contains user-readable AgentPal examples that may become future Wiki or docs material.

Use these examples to understand common AgentPal usage paths, first-team walkthroughs, direct Pal calls, handoffs, and PalSmith creation requests.

- `usage/`: common user-facing AgentPal usage examples.
- `quickstart/`: first-run examples for supported host entry paths.
- `walkthroughs/`: longer user-readable walkthrough examples.
- `pal-collaboration/`: readable collaboration examples.
- `palsmith/`: PalSmith creation and governance request examples.

Internal failure cases, boundary cases, regression materials, and model-behavior guard examples do not belong here.

Team Pack standard examples remain in `examples/team-packs/`.
