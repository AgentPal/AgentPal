# Pal Layer Vs Agent Layer

AgentPal is the Pal layer.

Pal layer:

- identity
- knowledge
- skills
- context
- memory boundary
- output contract
- coordination
- review
- summary
- learning

Execution runtime:

- reads files
- writes files
- runs commands
- calls tools
- performs system or network actions
- produces execution evidence

AgentPal helps the current runtime decide who should answer and what assets to use. It does not become the runtime itself.
