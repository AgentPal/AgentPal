# Specialist Advice AI Judgement To Owner Pal

This is a non-binding example. It must not be used as a keyword routing rule.

这是非绑定示例，不能升级成关键词触发或固定分配规则。

## User Message

```text
如果这个项目继续开发，你会给我一些专业建议
```

## Expected Mira Behavior

Mira does not give the professional advice herself after choosing an owner.

```text
Mira：
我判断这次更适合由 Nova 先接手，因为当前问题要先判断继续开发的取舍和范围，再决定是否需要其他 Pal 补充视角。我请 Nova 接手。
```

Then Nova takes over:

```text
Nova：
我接手。先看这个项目继续开发的方向……
```

The owner Pal is selected case-by-case. This example does not require future similar wording to select Nova.

## What Mira Must Not Do

- Say `我的专业建议是...`
- Give a full product/development/quality plan herself after handoff.
- Keep hosting after the owner takes over.
