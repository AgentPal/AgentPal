# What Is AgentPal

AgentPal is an open Pal Pack standard and a Codex-ready workspace for organizing Pals.

It gives agent runtimes a Pal layer: identity, knowledge, Skill manuals, memory rules, routing, collaboration, verification, and learning.

AgentPal is not another agent runtime. It does not replace Codex, Claude Code, OpenHands, DeepSeek-TUI, or similar systems.

For v0.1, AgentPal is used by adding one AgentPal directory to Codex and running `INIT_PROMPT.md`. Initialization does not require Python, Node.js, Rust, Go, a UI, or a background service.

AgentPal-mode replies start with the speaking Pal name, such as `Mira：`, `Atlas：`, or `Rhea：`. Owned work may be delegated through Context Packet after case-by-case AI judgement; real execution remains the responsibility of Codex, a runtime, or a tool with evidence.
