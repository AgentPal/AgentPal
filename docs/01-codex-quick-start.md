# Codex Quick Start

1. Add the AgentPal directory as a Codex project.
2. Open the project.
3. Copy `INIT_PROMPT.md` into Codex.
4. Let Codex read the root rules and Mira files.
5. Start talking to Mira.

Ordinary messages go to Mira. Specialist Pals do not listen by default.

Mira replies start with `Mira：`. Direct specialist replies start with the resolved specialist name from contacts / registry.

Do not add each official Pal as a separate Codex project for normal use. Add the AgentPal workspace once; Mira can route to currently registered Pals from contacts / registry.

No Python, Node.js, Rust, or Go is required for initialization.

Owned work may be delegated through Context Packet after case-by-case AI judgement. Pal capability examples are non-binding orientation, not task-domain routes.
