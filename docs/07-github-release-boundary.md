# GitHub Release Boundary

The public AgentPal repository should contain only release-safe Markdown, JSON, templates, and protocols.

Do not include private user memory, internal development records, copied reference docs, API keys, tokens, secrets, or private project data.

The current version is a Codex Workspace, not a desktop app. It should not include UI code, scripts, installers, runtime dependencies, or generated private state.

Third-party materials keep their original licenses. Do not copy third-party source or long text into the public repository.
