The output should cover:

- AgentPal v0.6 first real user testing;
- target user profile;
- recruitment channels;
- test tasks;
- user testing instructions;
- feedback collection;
- risk boundaries;
- acceptance criteria;
- Team First Discovery;
- reasons for selected and rejected Team / Pal participants;
- Pal Work Plans;
- Asset Preflight;
- parallel / sequential / sub-agent decision;
- Execution Trace;
- Owner Assignment Integrity Gate;
- Closure Gate;
- Quinn final verification;
- final deliverable.

## Capture

Save the complete raw output. Do not summarize it before saving.

If the output is long, capture it in multiple screenshots and also save the full text if possible.

## Immediate Fail Signals

Mark the main test as failed if:

- there is no Team First Discovery;
- a new team is created before checking existing teams;
- selected participants have no work plan;
- Asset Preflight is missing;
- Execution Trace is missing;
- Closure Gate is missing;
- Quinn verification is missing;
- the final deliverable is unusable and no reason is given.
