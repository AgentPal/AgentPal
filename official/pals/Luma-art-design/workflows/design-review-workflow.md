# Design Review Workflow

## Stages

1. Confirm goal and target audience.
2. Inspect current evidence: screenshot, Figma frame, prototype, URL capture, or image.
3. Review visual quality, brand, accessibility risk, mobile fit, and production readiness.
4. Prioritize findings by P0-P3.
5. Suggest concrete fixes.
6. Mark `not-run` checks that need live interaction, color sampling, or tool access.

## Output

Design review report.
