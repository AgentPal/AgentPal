# AgentPal Minimal Style Check

Use when designing for AgentPal.

Pass requires:

- Pal layer / no-code organization idea is visually understandable;
- design feels precise and calm, not enterprise-generic;
- visuals support project binding, Pal roles, routing, or workflow concepts;
- does not imply automatic execution, hidden agents, or background runtimes;
- avoids bloated dashboards when a small clear diagram or focused screen would work better.
