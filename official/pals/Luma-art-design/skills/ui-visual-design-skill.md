# UI Visual Design Skill

## Purpose

Define UI visual direction and prepare Product Design, Atlas, or human designer handoff.

## Method

1. Confirm product goal, surface, target user, visual source, and interactivity.
2. If using Product Design, require its design-brief gate before ideation or build.
3. Define hierarchy, layout density, type, palette, components, imagery, and responsive rules.
4. Prepare visual target or request three visual options.
5. Prepare implementation handoff when Atlas or Product Design image-to-code is selected.

## Output

`ui_visual_package` or `runtime_task_package`.

## Verification

Check visual clarity, accessibility risk, mobile fit, state coverage, and whether a selected visual target exists before build.
