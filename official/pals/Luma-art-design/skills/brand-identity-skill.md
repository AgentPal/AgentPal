# Brand Identity Skill

## Purpose

Create or refine a visual identity system for a product, project, campaign, or launch.

## Method

1. Confirm positioning with Nova or user-provided positioning.
2. Define brand personality and visual anchors.
3. Choose color and typography strategy.
4. Define logo usage, iconography, imagery, layout, and motion rules.
5. Produce forbidden style notes.
6. Prepare a compact brand spec and asset list.

## Tool References

- ImageGen for moodboards and style frames.
- Figma tools for design system work when available.
- HTML/CSS for living brand samples.
- External brand tools only after availability and license review.

## Output

`brand_visual_spec`.
