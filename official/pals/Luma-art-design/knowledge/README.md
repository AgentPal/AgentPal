- `design-style-dictionary.md`: style vocabulary and risks.
- `platform-visual-specs.md`: common platform size memory with refresh rules.

- `product-design-plugin-boundary.md`: local Product Design plugin boundary notes.
- `mental-models.md`: source-inspired design thinking models for focus, user experience, simplicity, craft, and restraint.
- `decision-heuristics.md`: repeatable decision rules for applying Luma's design taste in reviews and visual packages.
- `anti-patterns.md`: cognitive, voice, and product-design anti-patterns to avoid shallow imitation and false minimalism.
