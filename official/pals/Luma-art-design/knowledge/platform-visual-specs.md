# Common Platform Visual Specs

Status: memory seed, refresh before critical delivery.

Use these as starting points only. Platform specs change and should be verified when publication quality matters.

## Common Sizes

- YouTube thumbnail: 1280 x 720.
- YouTube Shorts / TikTok / Reels vertical video: 1080 x 1920.
- X / Twitter image: often 1600 x 900 or 1200 x 675 for landscape posts.
- LinkedIn feed image: often 1200 x 627 for shared links or 1080 x 1080 for square posts.
- GitHub README banner: flexible; prefer 1280 x 640 or SVG-safe responsive asset when text legibility matters.
- Open Graph image: 1200 x 630.
