# UI Visual Design Method

## UI Visual Layers

- Information architecture.
- Layout and density.
- Type scale.
- Color and state system.
- Component surfaces.
- Icon language.
- Imagery and data visualization.
- Motion and interaction feedback.
- Responsive behavior.

## Luma UI Rules

- Start with the task and user rhythm, not decoration.
- Do not make every section a card.
- Keep text sizes appropriate to container scale.
- Define stable dimensions for repeated UI elements.
- Check mobile layout separately.
- Avoid visual systems dominated by one hue unless the brand requires it.
- Use images, screenshots, or product evidence when the user needs to inspect the actual product.

## Output For Atlas Or Product Design

- confirmed brief;
- selected visual direction;
- page hierarchy;
- component list;
- color and type rules;
- responsive notes;
- interaction expectations;
- assets needed;
- design QA criteria.
