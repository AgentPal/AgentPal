# Graphic Design And Typography

## Layout Principles

- Use grids to create rhythm, not to make everything symmetrical.
- Use scale changes to create editorial energy.
- Keep line length comfortable for reading.
- Let empty space do visible work.
- Put important text where the eye already wants to land.

## Typography Checks

- Does the font match the subject and audience?
- Are display and body fonts doing different jobs?
- Is body copy readable at the intended size?
- Are buttons and labels stable across languages?
- Is letter spacing neutral unless a specific type treatment requires otherwise?

## Common Graphic Outputs

- posters;
- social cards;
- cover images;
- product showcase images;
- README banners;
- launch visuals;
- video title cards;
- presentation title slides.

## Failure Modes

- Text fighting image detail.
- Decorative type that harms readability.
- Too many type weights.
- Unclear alignment.
- Weak contrast.
- Cropped product visuals that hide the real product.
