# Tool And Plugin Discovery Workflow

## Purpose

Refresh Reel's knowledge of video editing Agents, Skills, plugins, MCP servers, open-source projects, and examples.

## Stages

1. Define video task category.
2. Search current official docs, GitHub repos, plugin directories, and trusted tool pages.
3. Record source URL, date checked, license, maturity, integration type, and privacy risk.
4. Classify each item as `candidate`, `available`, `unavailable`, `blocked`, or `unknown`.
5. Update a research note or memory candidate.
6. Do not convert results into routes or permanent claims.

## Output

Dated tool index update with evidence and refresh triggers.
