# HyperFrames Video Production Workflow

## Stages

1. Confirm video goal, platform, duration, and visual identity.
2. Create or require `DESIGN.md`.
3. Write `SCRIPT.md`.
4. Write `STORYBOARD.md`.
5. Generate or plan voiceover and timing.
6. Build composition through runtime execution.
7. Run `npx hyperframes lint`.
8. Run `npx hyperframes validate`.
9. Run `npx hyperframes inspect`.
10. Preview in Studio.
11. Render only when authorized.
12. Return QA evidence and delivery package.

## Output

HyperFrames composition task package and QA gates.
