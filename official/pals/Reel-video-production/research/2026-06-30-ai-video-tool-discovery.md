# AI Video Tool Discovery

Date checked: 2026-06-30

Purpose: initial research seed for Reel creation. This is not a permanent route map.

## Findings

- HyperFrames in the local Codex plugin is suitable for HTML video compositions, captions, TTS, audio-reactive visuals, animated text, scene transitions, validation, preview, and rendering. Local skill docs require visual identity before composition HTML and recommend lint, validate, inspect, preview, and render evidence.
- Remotion is a React-based framework for programmatic video, useful when videos are best represented as reusable components and code-driven renders. Source: https://www.remotion.dev/
- FFmpeg is a mature open-source multimedia toolkit for transcoding, trimming, filtering, metadata inspection, and audio/video processing. Source: https://ffmpeg.org/
- OpenMontage is an open-source agentic video pipeline candidate with stages such as research, proposal, script, scene plan, assets, edit, and compose. Source: https://github.com/calesthio/OpenMontage
- WhisperX is a word-level timestamp transcription candidate for subtitle sync workflows. Source: https://github.com/m-bain/whisperX
- Piper is a local neural TTS candidate. Source: https://github.com/rhasspy/piper
- Subtitle Edit is an open-source subtitle editing/conversion candidate. Source: https://github.com/SubtitleEdit/subtitleedit

## Reel Policy

- Record tool candidates with source, license, maturity, privacy risk, and availability evidence.
- Do not claim a tool was invoked unless there is current runtime evidence.
- Do not hard-code tool lists as permanent facts.
- Prefer HyperFrames for HTML/motion/caption/TTS compositions, FFmpeg for technical processing, Remotion for React video systems, and OpenMontage for agentic pipeline experiments when each is currently available and appropriate.
