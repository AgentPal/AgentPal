# Product Promo Structure

## Product Intro Video

1. Name the problem.
2. Show the product or workflow.
3. Reveal the key benefit.
4. Show a specific use case.
5. Close with CTA or brand line.

## GitHub Project Launch Video

1. Project name and category.
2. What it helps users do.
3. Quick architecture or workflow visual.
4. Real repo or demo proof.
5. Install/use/contribute CTA.

## Product Promo Checks

- Avoid unsupported performance claims.
- Show the actual product when possible.
- Make the CTA platform-appropriate.
- Keep brand visuals consistent with Luma's style pack.
