Status: memory seed, refresh before final delivery.

Use these as starting points only. Platform specs and app UI safe areas change.

## Common Formats

- YouTube standard video: 16:9, often 1920 x 1080.
- YouTube Shorts / TikTok / Instagram Reels: 9:16, often 1080 x 1920.
- Instagram feed square: 1:1, often 1080 x 1080.
- Instagram feed portrait: 4:5, often 1080 x 1350.
- Bilibili: commonly 16:9 for standard uploads; vertical formats also exist.
- Xiaohongshu: commonly supports vertical and square content; verify current recommendations.
- GitHub README video: prefer lightweight MP4/GIF/WebM depending repository policy and file size; often use externally hosted video or compressed demo clip.
