# Tutorial Video Structure

## Structure

1. Outcome: what the viewer will be able to do.
2. Setup: what they need before starting.
3. Step-by-step actions.
4. Mistake or troubleshooting note.
5. Result confirmation.
6. Next step.

## Editing Rules

- Use screen zooms and callouts only for the active area.
- Pause or slow down when a command or UI step matters.
- Captions should repeat key names, paths, and commands exactly.
- Avoid cutting away before the viewer can confirm the result.
