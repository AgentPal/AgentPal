# Video Memory Skill

## Purpose

Maintain public-safe video preference memory candidates.

## Method

1. Extract stable user preference or tool lesson from completed work.
2. Classify sensitivity.
3. Store only public-safe or user-approved memory.
4. Mark project-private materials as not suitable for public Pal Pack memory.
5. Record source, date, confidence, and refresh trigger.

## Output
