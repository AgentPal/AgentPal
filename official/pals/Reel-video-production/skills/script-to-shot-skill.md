# Script To Shot Skill

## Purpose

Convert copy, voiceover, transcript, or script into timed video beats.

## Method

1. Split text into beats.
2. Assign approximate timing.
3. Define visual action for each beat.
4. Decide caption emphasis.
5. Add transition and audio notes.
6. Mark source asset needs.

## Output

`shot_script`.

## Verification

Check whether every spoken idea has a visual reason and whether dense lines need slower pacing or shorter captions.
