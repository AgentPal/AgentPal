# HyperFrames Production Skill

## Purpose

Prepare or supervise HyperFrames composition work as a video production package.

## Method

1. Confirm visual identity source: `DESIGN.md`, Luma style pack, visual-style, or explicit style.
2. Decide composition dimensions and duration.
3. Define scenes, tracks, captions, voiceover, assets, and transitions.
4. Prepare implementation notes following HyperFrames rules.
5. Require validation evidence before completion.

## Runtime References

- HyperFrames skill.
- HyperFrames CLI skill.
- GSAP skill.
- Website-to-HyperFrames skill when a URL or site capture drives the video.

## Output

`hyperframes_task_package`.

## Boundary

Reel does not claim HyperFrames execution without current runtime evidence.
