# Research

## Purpose

This directory lists source materials, source inventories, investigations,
design notes, comparisons, evidence extracts, coverage matrices, and unverified
findings owned by PalSmith.

PalSmith research preserves source context for Pal creation, Pal upgrade, asset
classification, Pal governance, material ingestion, readiness review, and
research-to-knowledge planning before stable conclusions are promoted into
knowledge, standards, templates, runbooks, workflows, skills, evals, or memory.

## What Belongs Here

- Source inventories.
- Source coverage matrices.
- Investigation notes and design notes.
- Comparative research drafts.
- Evidence extracts and source-context notes.
- Public-safe research gap notes.
- Links to related knowledge, standards, templates, runbooks, workflows, skills,
  evals, reports, and memory candidates.

## What Does Not Belong Here

- Stable knowledge without source context.
- Long-term memory or durable governance lessons.
- Full task, health, readiness, import/export, or release reports.
- Runtime state or current mutable task state.
- Connector implementations, Runtime clients, scanners, validators, daemons,
  installers, databases, marketplaces, hubs, auto-sync engines, or
  auto-execution tools.
- Credentials, tokens, cookies, passwords, API keys, secrets, customer data, or
  private project evidence.
- External project `.agentpal/research/` copies by default.
- Keyword routing maps, domain maps, role maps, fixed Pal assignments, or
  deterministic dispatch tables.

## Promotion Path

Stable conclusions must be promoted only after review:

- stable PalSmith concepts -> `knowledge/`
- Pal creation or governance rules -> `standards/` or `core/`
- reusable report or task package shells -> `templates/`
- concrete operation manuals -> `runbooks/`
- multi-step normal processes -> `workflows/`
- Pal-owned methods -> `skills/`
- acceptance and regression checks -> `evals/`
- extracted long-term governance lessons -> `memory/`
- task completion evidence -> `reports/`

Preserve `unknown`, `not-run`, and `needs-review` for claims that have not been
verified.

## Relationship To Knowledge

Research is not stable knowledge. Knowledge should contain reviewed, stable,
de-identified conclusions and may point back to research when source context is
needed.

## Relationship To Memory

Research is not memory. Memory stores extracted long-term PalSmith governance
lessons after review. Raw source materials, investigations, design notes, and
unverified findings remain research until promoted.

## Public Safety Boundary

Do not include private sources, credentials, tokens, cookies, passwords, API
keys, secrets, customer identifiers, private project evidence, private user
memory, or copyrighted source excerpts beyond allowed use in public research
indexes.

## External Project Boundary

Do not copy PalSmith research into an external project `.agentpal/` directory by
default. External projects remain thin-bound; PalSmith research belongs in the
central Pal Pack or approved central records.
