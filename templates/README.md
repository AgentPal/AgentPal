# AgentPal Templates

This directory contains reusable public templates for AgentPal workspaces, Pal Packs, thin project bindings, task packages, context packets, memory records, and release-safe evidence.

Primary areas:

- `project-binding/`: thin external project binding templates.
- `Pal Pack/`: current Pal Pack templates kept for compatibility.
- `pal-pack/`: lowercase navigation pointer for Pal Pack templates.
- `organization/`: organization workspace template pointers.
- `governance/`: governance template pointers.
- `org-pack/`: org-pack template pointers.
- `prompts/`: prompt template pointers.
- `orchestration/`: Deep Conductor, Context Packet, verification, Runtime Skill, and context-budget task package templates.

Templates are Markdown / JSON assets only. They do not create a CLI, installer, scanner, daemon, service, or automatic execution layer.
