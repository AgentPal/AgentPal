# Base Org Pack

## Purpose

This Org Pack template describes a reusable no-code AI organization package for AgentPal.

It can be adapted into public-safe organization patterns, delivery packs, practice packs, or team-operation packages.

## Recommended Use

Use this template when an organization pattern should be reusable across projects while keeping customer-private facts out of the reusable package.

## Recommended Pals

Recommended Pals are not hard routes. They are AI judgement inputs only.

When using this Org Pack, the current Brain / AI should read the central roster, the active project context, the task, current evidence, and this Org Pack before choosing an owner Pal.

## Asset Boundary

Reusable assets belong in this Org Pack.

Customer-private assets belong in private project records or customer-private evidence records, not in this reusable pack.

## Runtime Boundary

This Org Pack does not execute work. Host runtimes perform real file, command, or external-system actions only when separately instructed and when evidence is available.
