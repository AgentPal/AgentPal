# Memory Policy

This Org Pack may describe reusable memory rules, but it must not include private memory.

Reusable memory rules can describe what kinds of project facts should be recorded, who should review them, and where they should live.

Project-specific memory belongs in the AgentPal central project record, such as:

```text
workspace/projects/<project-id>/memory/
```

External projects should not receive `.agentpal/memory/` by default.
