# Memory Protocol

公开发布包中不保存私人记忆。

## 可以记录

- 公开可复用的方法。
- 去敏后的任务模式。
- 用户明确允许保存的长期偏好。
- 重复任务候选。
- 知识缺口。

## 不可以记录

- 密钥、Token、账号、密码。
- 私人聊天全文。
- 客户资料。
- 真实项目敏感状态。
- 未经用户允许的长期记忆。

## Skill 触发

用户明确要求保存 Skill，或类似任务超过 3 次时：

1. 优先在本 Pal `skills/` 下创建正式 Skill。
2. 更新 `skills/index.md`。
3. 如果信息不足或有风险，先写入 `learning/` 候选区。
