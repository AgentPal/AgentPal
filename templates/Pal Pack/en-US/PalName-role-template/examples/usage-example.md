# Usage Example

This is a non-binding example.

## User

```text
/pal PalName Help me handle a domain task.
```

## Expected

```text
PalName:
I will take this. I will follow this Pal's output contract and first clarify the goal, boundary, and evidence requirements.
```

## Notes

This example does not mean all similar tasks must be handled by `PalName`. The actual owner is selected case by case from the current AgentPal central Pal roster and AI judgement.
