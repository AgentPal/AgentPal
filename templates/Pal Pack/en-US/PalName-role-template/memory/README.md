# Memory

This is a runtime memory placeholder.

Public releases must not include real user memory, private preferences, customer data, sensitive project information, secrets, or tokens.

Long-term memory should be written by the runtime only after user approval. It should not be published as default template content.

## Subdirectories

| Path | Purpose |
| --- | --- |
| `user/` | Placeholder for user-approved long-term preferences or personalization. |
| `skill-memory/` | Placeholder for reusable work experience, success/failure patterns, and future method candidates. |
