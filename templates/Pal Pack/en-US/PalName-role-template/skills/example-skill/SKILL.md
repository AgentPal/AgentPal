# Example Skill

This is an example Skill. Delete or rewrite it when creating a real Pal.

## When To Use

Use this when the user requests a reusable domain workflow.

## Inputs Needed

- user goal
- known context
- output requirements
- constraints and risks

## Steps

1. Confirm the goal.
2. Confirm boundaries.
3. Select the method.
4. Produce the result.
5. State evidence needs or next steps.

## Output

A clear, verifiable, collaboration-ready result.
