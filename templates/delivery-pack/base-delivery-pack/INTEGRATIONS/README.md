# Integration Notes

Use this directory for public-safe notes about business systems and access assumptions.

This directory does not contain connectors, API clients, credentials, tokens, cookies, passwords, webhook secrets, scanner logic, validator programs, or automatic discovery code.

## Integration Note Shell

```text
system_name:
business_purpose:
access_status: unknown
credential_storage: forbidden in reusable pack
connector_execution: not-run
data_sensitivity:
human_review_needed:
private_record_target:
```

For real projects, store private access details only in approved private project records.
