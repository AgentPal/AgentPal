# Reports

Use this directory for report shells and public-safe report templates.

Real customer delivery reports, private evidence, private project memory, and sensitive business outcomes belong in approved private project records unless explicitly reviewed and de-identified.

## Report Shell

```text
report_id:
report_type:
project_ref:
flow_ref:
task_refs:
summary:
evidence:
open_risks:
human_review:
private_writeback_target:
status:
```
