# Delivery Pack

## Purpose

Describe the delivery goal, intended users, and how this pack should be used.

## Delivery Position

This pack follows the AgentPal Delivery Pack structure:

```text
Project
  -> Flow Pack
  -> Task Package
```

## Included Assets

- Project context: `PROJECT.md`
- Flow Pack shells: `FLOWS/`
- Task Package shells: `TASKS/`
- Pal Team Blueprint: `PAL_TEAM.md`
- Integration notes: `INTEGRATIONS/`
- Acceptance criteria: `ACCEPTANCE.md`
- Training notes: `TRAINING.md`
- Report shells: `REPORTS/`
- Faye to PalSmith handoff: `FAYE_BUILD_REQUEST.md`

## Not Included

- credentials;
- customer-private assets;
- connector execution;
- scanner or validator programs;
- marketplace or installer behavior;
- automatic runtime execution;
- central roster or official Pal changes;
- external project `.agentpal/` writes by default.
