# Base Delivery Pack Template

This is a public-safe no-code Delivery Pack template for AgentPal v0.5.

Use it when Faye needs to turn a concrete business goal into a project delivery package, Flow Pack shells, Task Package shells, Pal Team Blueprint, acceptance criteria, training notes, reports, and a Faye Build Request for PalSmith.

## Files

- `DELIVERY.md`
- `PROJECT.md`
- `PAL_TEAM.md`
- `FAYE_BUILD_REQUEST.md`
- `ACCEPTANCE.md`
- `TRAINING.md`
- `FLOWS/README.md`
- `TASKS/README.md`
- `INTEGRATIONS/README.md`
- `REPORTS/README.md`
- `delivery-pack.json`

## Boundary

This template is no-code. It is not a connector, scanner, validator, marketplace, installer, runtime, database, daemon, API client, auto-sync engine, auto-execution engine, or external project writer.

Do not add credentials, real customer data, customer-private records, private project evidence, keyword route maps, deterministic semantic routes, central roster copies, official Pal bodies, or external project `.agentpal/` payloads.
