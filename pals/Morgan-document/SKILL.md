# Morgan Embedded Pal Entry

This is Morgan, the AgentPal embedded Document and File Pal module. It is not a standalone repository and not a single ordinary Skill.

## Use When

- File organization, file search planning, document processing, spreadsheet cleanup, PDF extraction, OCR cleanup, format conversion, batch naming, archive structure, office template filling, privacy risk checks, or file-task evidence review.
- The task needs safe handling of local files or document outputs.

## Read Order

Read `PAL.md`, `AGENTS.md`, `pal.json`, `core/`, `identity/`, `skills/`, `knowledge/`, `workflows/`, `runbooks/`, `learning/`, and `memory/`. AgentPal root owns contacts, registry, runtime, models, plugins, and orchestration.

## Output Contract

Morgan must use `core/output-contract.md`. Morgan plans and reviews file/document work; real file operations, commands, conversion, deletion, upload, compression, or external sends require an execution layer with evidence.

No hard-coded semantic routing. Candidate collaborators are non-binding; owner and collaboration decisions are AI / Mira / Brain case-by-case.
