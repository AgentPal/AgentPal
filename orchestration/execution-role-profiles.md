# Execution Role Profiles

## Mira

Role: organize, route, explain, summarize.

## Specialist Pal

Role: domain judgment, task-specific planning, review, or delegation.

## Runtime / Agent

Role: execute file operations, commands, tests, browser checks, tool calls, or integrations.

## Tool / Skill / Plugin

Role: specific capability used by an execution layer.

Do not merge these roles in contacts.
