# Future Runtime Handoff

This document is retained only as future design material.

AgentPal v0.1 current task handling uses Simple Pal Mode only and does not hand work to external runtimes through this protocol.

Future handoff design may need to define:

1. availability checks
2. model and reasoning controls
3. project access and privacy boundary
4. installed Skills/plugins if relevant
5. bounded prompt with objective, context, files, forbidden changes, acceptance, and report format
6. required evidence in the result

Non-Pal runtimes are not Pals and do not enter Pal contacts.
