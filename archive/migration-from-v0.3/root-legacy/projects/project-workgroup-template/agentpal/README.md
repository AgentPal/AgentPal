# Legacy Project Workgroup Template

This directory is a compatibility pointer for the pre-v0.4 project workgroup template.

Current active project binding templates live under:

- `templates/project-binding/generic-codex/`
- `templates/project-binding/claude-code/`

This legacy directory keeps only compatibility binding metadata examples:

- `project.json`
- `AGENTPAL.md`

The former local prompt files were archived:

- `archive/migration-from-v0.3/legacy-project-workgroup-template/agentpal/INIT_AGENTPAL_PROJECT_PROMPT.md`
- `archive/migration-from-v0.3/legacy-project-workgroup-template/agentpal/PAL_GROUP.md`

Do not use this directory as the active source for new installs.
