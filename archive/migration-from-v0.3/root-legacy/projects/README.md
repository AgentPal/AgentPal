# Legacy Projects

This archived copy records old external project binding pointers and templates that previously lived at root `projects/`.

Project means external user project by default, not the AgentPal workspace itself.

Use `templates/project-binding/generic-codex/` or `templates/project-binding/claude-code/` as the current source template. Active project-binding protocols now live under `standards/project-binding/`. Legacy `project-workgroup-template/agentpal/` remains a compatibility pointer only and is not the active install template.
