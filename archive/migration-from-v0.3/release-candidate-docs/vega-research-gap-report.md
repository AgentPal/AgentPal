## PalSmith-Style Fitness Conclusion

Vega already has a valid research Pal identity, a source-aware tone, source quality notes, and several older research skills. The role is not empty. The gap is that the assets still read like a research support Pal rather than an official Research / Intelligence Lead Pal. The old structure has useful directory skills, but it does not yet give Vega a complete end-to-end research operating system: research-question-framing, search-strategy-design, source-inventory, source-coverage, evidence-matrix, claim-evidence-alignment, research-synthesis, comparative-analysis, uncertainty-and-confidence-reporting, knowledge-distillation, and research-handoff need explicit skill cards, knowledge cards, workflows, runbooks, and evals.

## Current Problems

- Role Fit: Vega is described as a research analysis Pal, but not consistently as the official Research / Intelligence Lead Pal.
- Research Expertise Depth: existing source and report assets are useful, but they do not form a complete job model.
- Source Inventory Discipline: no dedicated source inventory and coverage matrix assets existed before this round.
- Evidence Matrix Building: old evidence table material exists, but claim, source, confidence, contradiction, and open question structure is not consistently required.
- Claim-Evidence Alignment: the current identity says to separate fact and inference, but it lacks a named skill and eval.
- Synthesis Quality: report writing exists, but synthesis rules are not yet strong enough for fact, inference, recommendation, uncertainty, and user decision boundaries.
- Comparative Analysis: existing comparison skills need connection to evidence quality and source coverage.
- Collaboration Fit: collaboration language exists, but Vega needs a default-Pal boundary file showing consult, delegate, handoff, shared context, withheld context, and expected output.
- No-Code Boundary: current files say Vega is not the executor; the upgraded role must keep this boundary while becoming deeper.
- Evaluation: existing evals are broad release checks; dedicated source credibility, evidence matrix, synthesis, knowledge distillation, and self-health evals are missing.
