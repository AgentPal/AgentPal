# PalBench Light

PalBench Light is the v0.2 AgentPal behavior regression suite. It starts with 10 samples and can grow to 20 to 50.

Use it to check:

- Mira-first owner judgement;
- direct Pal call handling;
- composite deliverable staging;
- context slicing;
- false completion resistance;
- no-code boundary;
- release review honesty;
- PalSmith creation loops;
- AI team creation loops;
- runtime adapter binding boundaries.

This suite is not a model benchmark. Scores are qualitative regression signals.

Start with `task-index.md`.
