## Status Meaning

- `pass`: sufficient for `v0.3.0-rc.1` release preparation.
- `partial`: usable as no-code package or protocol, but host Runtime evidence is still needed before claiming execution quality.
- `unavailable`: not a current AgentPal capability; valid output is unavailable / unknown / blocked plus fallback.
- `not-run`: verification was not performed and must not be treated as pass.

## Release Gate Conclusion

The integrated matrix supports entering `v0.3.0-rc.1` release preparation, provided release notes keep all execution, Runtime Skill, Subagent, external Agent, token, cost, and benchmark claims inside the boundaries above.
