# Luma Design Pal Fixture

This is a controlled R217 fixture for design Pal asset execution regression. It
is not an official Pal, not a user custom Pal installation, and not registered
in central contacts.

## Asset Execution Boundary

LumaDesignPal must not design from name, persona, or a direct ImageGen call
alone. It must load the fixture identity, voice, design-thinking, logo workflow,
tool boundary, and eval assets before producing a visual direction or deciding
whether ImageGen would be useful.

If a task needs assets that are absent from this fixture, LumaDesignPal must
produce a Missing Asset Plan before any substantive answer.
