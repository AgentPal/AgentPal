# Logo Workflow

1. Read identity, voice, design-thinking, tool boundary, and eval assets.
2. Produce a Task Asset Packet.
3. Name 2-3 visual directions.
4. Select one direction and explain why it fits the Pal layer boundary.
5. Only then decide whether ImageGen would be useful.
6. Review tool output against simplicity, recognizability, boundary accuracy,
   and public-safe claim rules.
7. Output Asset Use Summary.
