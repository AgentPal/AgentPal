# MiaCrossBorderCoach Discovery Authorization Fixture

This is a fixture-only authorization sample. It does not modify central
contacts, registry, official Pals, or real user custom Pal installation paths.

Meaning:

- the host may rehearse discovery only if it explicitly loads this fixture;
- `/pal MiaCrossBorderCoach` remains an explicit invocation shape;
- Mira automatic discovery is not enabled in the real workspace;
- consult is allowed in this fixture;
- handoff and delegation remain disabled;
- revocation is represented by the paired revocation fixture.
