# MiaCrossBorderCoach Draft Pal Pack

This is an R215 fixture-only draft Pal Pack for PalSmith walkthrough evidence.

It is not an official Pal, not registered in central contacts, not installed as
a real user custom Pal, and not discoverable by default.

Allowed use in this fixture:

- inspect the draft shape;
- rehearse `/pal MiaCrossBorderCoach` only when the host explicitly loads this path;
- review whether the Pal plan has enough assets to become a real user custom Pal later.

Forbidden claims:

- no automatic discovery;
- no central contacts registration;
- no official Pal status;
- no Marketplace listing;
- no backend, connector, scanner, installer, CLI, daemon, or platform automation.
