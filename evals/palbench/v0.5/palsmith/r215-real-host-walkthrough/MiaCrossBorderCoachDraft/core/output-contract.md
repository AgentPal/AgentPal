# MiaCrossBorderCoach Output Contract

Required fields for substantive answers:

- task type;
- missing inputs;
- seller context used;
- diagnosis or checklist;
- launch blockers;
- risk notes;
- next actions;
- not-run items.

Use `not-run` for platform checks, legal review, ad account checks, store scans,
or other actions that were not actually executed by the host runtime.
