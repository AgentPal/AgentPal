# First-Principles Product Reviewer

This is a PalSmith draft Pal Pack test artifact, not an official Pal.

Status: `draft_test_artifact`

This draft exists for R174 to verify that PalSmith can move from a creation plan to actual Pal Pack files in a non-official test directory.

It is not registered in central contacts, not listed as an official Pal, and not ready for public product release. It should not be published as a marketplace item without a separate source, quality, privacy, and publication review.

## How To Read This Draft

- `PAL.md` defines the draft Pal identity and work boundaries.
- `SKILL.md` is the draft entry point for loading the whole pack.
- `pal.json` records machine-readable draft metadata.
- `identity/` defines role, source boundary, and tone.
- `knowledge/` keeps the product-review mental models and job knowledge.
- `workflows/` contains review and complexity-compression flows.
- `skills/` maps Pal-owned skills and host Agent capability candidates.
- `runtime/` records when the host runtime may be asked to act.
- `memory/` defines approved memory candidate shapes.
- `contacts/` records collaboration boundaries.
- `evals/` defines the draft quality gate.
- `marketplace/` contains metadata and publication boundary drafts only.

## Boundary

This draft is public-source-inspired in method, not a real-person representation. It does not copy external source files, does not claim authorization from any person or organization, and does not include runtime code, CLI, scanner, daemon, connector, backend service, or Marketplace backend.
