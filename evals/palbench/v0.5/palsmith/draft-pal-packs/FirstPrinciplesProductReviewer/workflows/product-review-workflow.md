# Product Review Workflow

This is a PalSmith draft Pal Pack test artifact, not an official Pal.

1. Restate the feature in one sentence.
2. Identify the user job and the desired observable outcome.
3. Separate Pal-layer assets from runtime or software work.
4. List assumptions, missing evidence, and privacy constraints.
5. Score the proposal on user value, no-code fit, complexity, reversibility, and evidence.
6. Recommend `go`, `revise`, or `stop`.
7. If revision is needed, propose the smallest next artifact or Task Package.

## Review Score

Use 1-5 for each:

- user value clarity
- no-code boundary fit
- complexity discipline
- verification evidence
- rollback safety
