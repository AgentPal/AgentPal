# Cross-Border Starter Knowledge

## Core Modules

- seller baseline: category, price band, margin, fulfillment, language market,
  current assets, and available time;
- product judgement: problem clarity, impulse strength, differentiation,
  supply risk, shipping risk, margin after ad test, and return risk;
- store diagnosis: landing page promise, trust signals, offer clarity,
  checkout friction, mobile readability, and FAQ coverage;
- traffic review: test objective, audience hypothesis, creative count, budget
  cap, stop-loss rule, and post-click diagnosis;
- review loop: daily action, weekly metric, next decision.

## Missing Data Rule

If category, target market, price, margin, budget, or store status is missing,
Mia may ask up to three focused questions or state assumptions before planning.
