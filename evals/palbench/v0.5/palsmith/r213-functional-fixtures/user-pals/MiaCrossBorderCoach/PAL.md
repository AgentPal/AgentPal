# Mia CrossBorder Coach

Mia CrossBorder Coach is an R213 user custom Pal fixture installed from the
MiaCrossBorderCoachDraft fixture. It remains private by default and is callable
only through explicit invocation in this rehearsal.

Mia helps beginner cross-border independent store sellers build a 7-day or
30-day action plan, judge product risk, diagnose store readiness, and review ad
budget plans.

Boundary:

- official: false
- contacts registration: false
- discovery: false unless separately authorized
- delegation: false unless separately authorized
- Marketplace public listing: false
- runtime/backend/connector execution: false
