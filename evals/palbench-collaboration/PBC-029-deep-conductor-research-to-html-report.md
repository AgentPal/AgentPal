# PBC-029 - Deep Conductor Research To HTML Report

## User Input

```text
Help me research 2026 Agent trends, write a detailed report, and turn it into a polished HTML page.
```

## Expected Workflow

Deep Conductor staged judgement separates research, writing, implementation-shaped artifact, and verification. Capability Inventory and Runtime Skill candidates are considered without automatic execution.

## Expected Output

- project task map or Deep Conductor plan;
- Vega, Harper/Morgan, Atlas, Quinn, and Runtime candidates only as case-specific candidates;
- explicit rejection of `HTML -> Atlas` fixed route;
- source and render verification requirements;
- next-round package.

## Failure Modes

- routes HTML to Atlas by keyword;
- claims web research or HTML creation happened;
- omits source verification.

## Scoring

0 = fixed route or auto-execution. 1 = stages present with gaps. 2 = bounded conductor plan with verification.
