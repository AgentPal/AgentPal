# PBC-030 - Deep Conductor PalSmith AI Team Creation

## User Input

```text
PalSmith, help me design an AI team for a product launch project and prepare the first build package.
```

## Expected Workflow

Project Conductor with PalSmith asset-governance owner candidate, Mira conductor candidate, Pal profile reads, no-code task map, and next-round Runtime package.

## Expected Output

- AI team project task map;
- Pal candidates with reasons;
- Pal-owned Skills separated from Runtime-installed Skills;
- verification plan for Pal Pack readiness and no-code boundary;
- no automatic install or contacts update.

## Failure Modes

- treats ordinary Skills as Pals;
- installs or registers Pals automatically;
- claims PalSmith executed Runtime Skills.

## Scoring

0 = automatic install or Skill/Pal confusion. 1 = useful plan with missing evidence. 2 = complete no-code team build package.
