# GitHub Release Boundary Check

## Release files

- [ ] MIT License exists.
- [ ] README is available in English and Chinese.
- [ ] CONTRIBUTING exists.
- [ ] CHANGELOG exists.
- [ ] THIRD_PARTY_NOTICES exists.

## Boundaries

- [ ] no internal dcos content copied into release files
- [ ] no real user data
- [ ] no real private project data
- [ ] no API Key / Secret / Token
- [ ] no runtime dependency
- [ ] no Python / Node / Rust required
- [ ] no UI code
- [ ] no script tools
- [ ] no installers

## Third-party notices

- [ ] Third-party resources keep original licenses.
- [ ] Long third-party source or documentation is not copied.
