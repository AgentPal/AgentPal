# AgentPal v0.5 Release Notes Draft

Status: draft only. This is not a published release note.

## Summary

AgentPal v0.5 expands the public no-code organization layer around Pal teams, Pal assets, Org Packs, FDE Packs, combined pack examples, PalSmith governance, and release-quality evidence. It keeps the runtime boundary intentionally narrow: AgentPal is a Markdown / JSON / protocol workspace read by host runtimes, not an execution layer.

## Highlights

- Adds public Org Pack standard, practical Org Pack template, and Org/FDE combined usage guidance.
- Adds FDE Pack base template and public-safe combined pack example.
- Adds Asset Boundary standards to separate reusable public assets from private customer or project assets.
- Adds Pal Asset standards and Pal Metadata v0.5 upgrade planning.
- Adds PalSmith v0.5 creation, classification, template, README, and research safety guidance.
- Adds R127-R131 v0.5 regression, completion, and release-preflight evidence.
- Keeps Simple Pal Mode as the current runtime policy.

## Explicit Non-Goals

- No automatic multi-agent execution.
- No active Deep Conductor runtime.
- No external Agent calls.
- No runtime scanner, validator, installer, connector, or marketplace program.
- No keyword route map.
- No bundled private customer assets.
- No remote publication evidence in this draft.
