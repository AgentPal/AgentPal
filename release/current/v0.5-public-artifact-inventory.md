## Excluded From Public Package

- Private operator notes and local completion reports stored outside the repository.
- Customer-private project materials.
- Credentials, tokens, API keys, or secrets.
- Runtime executables or required new dependencies.
- Marketplace, connector, installer, scanner, validator, or external Agent integration code.
