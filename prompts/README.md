# Prompts

`prompts/` contains copy-and-paste prompts for using AgentPal in Codex.

These files are not programs. They do not install dependencies, run scripts, scan the disk, or create UI.

Use them when you want Codex to initialize AgentPal, add a Pal, refresh the Pal index, bind AgentPal to an external project, or rebuild Mira from public workspace files.
