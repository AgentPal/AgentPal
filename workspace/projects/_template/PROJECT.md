# Project Record Template

This directory is the central AgentPal record for one external user project.

It is not copied into the external project by default. The external project keeps only a thin binding that points back to this record.

Record:

- project display name
- project id
- active project root
- AgentPal workspace root
- `agentpal_project_record`
- binding style
- source map
- derived knowledge
- project memory
- task ledger
- governance notes
- capability inventory
- verification and report links
