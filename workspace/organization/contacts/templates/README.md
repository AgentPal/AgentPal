# Contact Templates

This directory contains templates for central Pal contact assets.

Contact templates are for Pal records only. Do not use them for ordinary Skills,
tools, plugins, models, MCP servers, knowledge packs, persona packs, raw
repositories, or runtime adapters.

Use `pal-capability-card.template.json` when adding or upgrading a Pal contact
so Mira and owner Pals can make case-specific owner judgements with both
positive capability signals and negative boundary checks.

Capability cards are judgement inputs only. They are not keyword routes.
